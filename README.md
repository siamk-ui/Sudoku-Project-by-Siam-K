Project Description
The purpose of this generator is to be able to generate and display a fully solved 9×9 Sudoku board. Sudoku is a number-placement puzzle where each number from 1 to 9 must appear exactly once in every row, column, and 3×3 subgrid. The program strats off by creating an empty 2D array to represent the board, in which all the values start off with 0 and  then uses a recursive backtracking algorithm to fill the board with valid numbers while following all Sudoku rules. Once a valid solution is found, the completed board is printed in a structured format that visually separates it into those 3 by 3 sections

How to Run the Program

You have to first open JavaSandbox and then create a Java file. Once you do you have to rewrite the main file name into SudokuRandomGenerator.java. Then You paste the source code. Then you simply click Run and wait for the program to generate

How the board is generated

The Sudoku board is generated using a backtracking algorithm that fills the grid one cell at a time. The program scans the board for an empty cell and tries placing numbers from 1 through 9, checking each placement with a validation function to ensure it does not violate Sudoku rules

Files Included

The main file is the source code itself, which, when pasted in Java Sandbox, runs the code without any additional materials needed.
